# Multithreaded Recursive Merge Sort

Recursive merge sort with POSIX threads

### How to run

##### Mergesort

`> clang++ -lpthread mergesort.cpp -o mergesort`  
`> ./mergesort`  

##### Benchmark

`> clang++ main.cpp benchmark.cpp -o benchmark`  
`> ./benchmark`  

### Requirements

* C++ 98 (or higher)
* POSIX threads
